# Honolulu, HIThis is a repo for managing machine readable artifacts for the [Honolulu, HI APIs](https://data.honolulu.gov/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Honolulu, HI APIs](https://data.honolulu.gov/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://data.honolulu.gov/)- [Developer](https://data.honolulu.gov/)The [APIs.json](https://github.com/api-evangelist/honolulu-hi/blob/master/apis.json) index provides links to all the relevant links for the Honolulu, HI APIs, including the OpenAPI and Postman if they are present.
